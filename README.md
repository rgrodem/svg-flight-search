# SVG Flight Search

En enkel webapplikasjon for å søke etter flyavganger og -ankomster fra/til Stavanger Lufthavn (SVG).

## Bruk

Besøk: https://rgrodem.github.io/svg-flight-search/

## Funksjonalitet

- Søk etter flyvninger for en valgt tidsperiode
- Filtrer basert på spesifikke flyplasser
- Se både avganger og ankomster
- Sanntidsdata fra Avinor API

## Teknologi

- Vanilla JavaScript
- Tailwind CSS
- Avinor API
- GitHub Pages

## Data

Data hentes i sanntid fra Avinor sitt offisielle API.
